SensorDB: an Open-source Platform for Big Sensory Data

Existing relational database suffers deteriorated performance as the volume of data scales due to the maintainance overhead of exponentially increasing indexing sets of existing data. While the noSQL database is not able to efficiently serve the spatial-temporal indexing which is critical for a large portion of sensory data.

We developed SensorDB, a semi-relational database to house very large spatial-temporal indexed sensory data sets. The software is made publicly available as an open-source software.
