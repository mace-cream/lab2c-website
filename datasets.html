Datasets

  1. Home
  2. Datasets

We are pleased to open the access to the datasets that we collected from our deployments of sensor network systems, and hope this will help the researchers and application developers in the related areas including sensor networks, cyber physical social systems, distributed signal and information processing, and applications of the Internet of Things.

The currently available datasets are Beijing taxi trajectory datasets. The datasets consist one-month trajectory data of 28,000 taxicabs in Beijing, which are approximately 42% of the total Beijing taxi (about 67,000). The total automobiles in Beijing is 5 million, and the datasets provide a pretty good sub-sampling set of the totality. On the time aspect, the dataset covered weekdays, weekends, and public holidays, being able to reflect mobility patterns in heavy, moderate, and light traffic conditions.

We hope the sharing of the real world sensory dataset will enrich the resources of the community, and would appreciate your acknowledgement in the publications and the citation to the following relevant publications.

    Visualization

    Description

    The data are segmented into days. To decrease the size of the files, the data are formatted into a specialized binary format. The files consist of multiple serialized data units of the following structure, and each data unit is a sample of a taxi's location and time.

    Please note that int is 32-bits and the structure is 28 bytes in length. Different C/C++ compilers may have different interpretations of int and the sizeof() function. Except this, the structure definition is pretty self-explainatory.

    Data Structure

                    typedef struct _data_unit{
                    int CUID;    // Car ID  (unique in all files)
                    int UTC;     // UTC time  (Seconds elapsed since 1970-01-01 00:00 GMT)
                    int LAT;     // Location Latitude  (Latitude*10e5)
                    int LONG;    // Location Longitude (Longitude*10e5)
                    int HEAD;    // Orientation (0-360)
                    int SPEED;   // Speed (cm/sec)
                    int OCCUPANT;   // The taxi is occupied(1) or empty(0)
                } data_unit;
