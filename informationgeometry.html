Information Geometry Based Analyses and Applications of Machine Learning Algorithms

  1. Home
  2. Information Geometry Based Analyses and Applications of Machine Learning Algorithms

Introduction

Based on the information geometry method from information theory, we build a theoretical framework for machine learning algorithms. With this framework, we further conduct performance analysis and provide interpretations of machine learning algorithms. The corresponding theoretical results will be then used to improve the performances of current algorithms and design new algorithms.
